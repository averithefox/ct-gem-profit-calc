## Installation

1. Download the latest release from the [Releases page](https://github.com/averithefox/ct-gem-profit-calc/releases)
2. Extract the `gem-profit-calc.zip` (or `gem-profit-calc-vX.X.X.zip`) file
3. Copy the extracted folder to ChatTriggers' modules directory (`.minecraft/config/ChatTriggers/modules`)
4. Reload ChatTriggers modules (`/ct reload` in-game)
